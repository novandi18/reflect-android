  Reflect is an Android application designed to help users track their daily moods, maintain a personal journal, and gain valuable insights into their emotional well-being. Through an intuitive interface, users can log their daily moods with intensity levels, record personal reflections, and analyze patterns over time. The app provides AI-powered insights to help users understand their emotional triggers and trends, promoting better mental health awareness and self-reflection. Built with [Jetpack Compose](https://developer.android.com/jetpack/compose).

</div>

## Features
- **Daily Mood Tracking:** Log your mood with intensity levels (1-10) and identify triggers
- **Personal Journal:** Write private reflections with custom tags and categories
- **Smart Insights:** AI-powered analysis of mood patterns and emotional trends
- **Mood History:** Visual calendar view and detailed history of past entries
- **Weekly & Monthly Summaries:** Comprehensive reports on mood distribution and patterns
- **Secure Storage:** PIN and fingerprint protection for private entries
- **Smart Reminders:** Daily notifications to maintain consistency
- **Dark Mode Support:** Comfortable viewing in any lighting condition

- [Firebase Authentication](https://firebase.google.com/docs/auth) - User Authentication

- [DataStore](https://developer.android.com/topic/libraries/architecture/datastore) - User Preferences

- [Material Design 3](https://m3.material.io/) - Design System
- [Navigation Component](https://developer.android.com/guide/navigation) - App Navigation
